# ARB (Zestaw zasobów aplikacji)

Languine obsługuje pliki ARB, standardowy format lokalizacji dla aplikacji Flutter. Ten format obsługuje liczby mnogie, miejsca na tekst oraz wyrażenia wyboru, zachowując zgodność z pakietem intl Fluttera.

---

## Konfiguracja

Najpierw upewnij się, że masz plik konfiguracyjny languine.json w katalogu głównym projektu. Oto przykład:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "arb": {
      "include": ["lib/l10n/intl_[locale].arb"]
    }
  }
}
```

## Tłumaczenie

Po ustawieniu konfiguracji uruchom:

```bash
npx languine@latest translate
```

Kiedy uruchomisz to polecenie, Languine:

- Załaduje twój źródłowy plik ARB (np. lib/l10n/intl_en.arb)
- Wykryje wszelkie nowe lub zmodyfikowane ciągi tłumaczeń
- Wygeneruje tłumaczenia dla twoich docelowych języków
- Utworzy lub zaktualizuje pliki ARB dla języków docelowych (np. lib/l10n/intl_sv.arb)
- Zachowa metadane, miejsca na tekst oraz liczby mnogie/wyrażenia wyboru
