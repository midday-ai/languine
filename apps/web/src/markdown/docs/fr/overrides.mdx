# Remplacements

Languine vous permet de remplacer les traductions pour des clés spécifiques. Cela est utile lorsque vous souhaitez vous assurer qu'une traduction particulière est utilisée pour une clé donnée, indépendamment de la traduction par défaut.

## Comment cela fonctionne

Dans le tableau de bord, vous pouvez créer des remplacements pour des clés spécifiques. Vous pouvez également créer des remplacements pour des langues spécifiques.
