# Sobrescrituras

Languine te permite sobrescribir traducciones para claves específicas. Esto es útil cuando deseas asegurarte de que se utilice una traducción específica para una clave particular, independientemente de la traducción predeterminada.

## Cómo funciona

En el panel de control, puedes crear sobrescrituras para claves específicas. También puedes crear sobrescrituras para idiomas específicos.
