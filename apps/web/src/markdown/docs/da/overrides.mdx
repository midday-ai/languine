# Overskrivninger

Languine giver dig mulighed for at overskrive oversættelser for specifikke nøgler. Dette er nyttigt, når du vil sikre, at en bestemt oversættelse anvendes for en bestemt nøgle, uanset den standardoversættelse.

## Sådan fungerer det

I dashboardet kan du oprette overskrivninger for specifikke nøgler. Du kan også oprette overskrivninger for specifikke sprog.
